# ✅ To-Do List

## About the Project

This To-Do List is a simple and responsive web application built using **HTML, CSS, and JavaScript**. I created this project to improve my understanding of JavaScript by building a practical application that helps users organize and manage their daily tasks efficiently.

The application allows users to add, complete, and delete tasks with an intuitive interface. Completed tasks are moved to a separate section, and all tasks are stored using **Local Storage**, ensuring they remain available even after refreshing the page.

## ✨ Features

- Add new tasks
- Mark tasks as completed
- Separate pending and completed tasks
- Delete tasks
- Data persistence using Local Storage
- Responsive and user-friendly interface

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
